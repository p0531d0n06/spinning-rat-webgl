# spinning-rat-webgl
Made the meme spinning rat in WebGL as a learning experience.

To use the app
- Download all the files into a folder
- Load the folder in VSCode
- Use the Live Server extension (https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) and click the "go live" button.
- You have your very own spinning rat!
- In the inspector console, you can set the "ySpin" and "xSpin" to your own values to spin the rat in different directions!
